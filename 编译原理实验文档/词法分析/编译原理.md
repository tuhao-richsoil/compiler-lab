## 二、注意点（转换图主要关注这）

+ `+ += ++`；`- -= --` 
+ `* *= */`
+ `/ // /* /=`
+ `< <= << <<=`；`> >= >> >>=`
+ `= ==`
+ `! !=`
+ `& && &=`；`| || |=`

## 三、难点

+ 中文字符识别与输出
+ `*`作为指针定义、乘号、指针解引用的区分
+ `< >`
+ `#include #define`作为宏定义整体识别
+ 常量类型细分
+ 新增字符常量识别，如`'a'`
